= NeoRadiant Script Reference
:toc: left
:toclevels: 3
:icons:
:iconsdir: img

NeoRadiant offers a *Lua* scripting interface which provides access to the most
important editor functions, including the Virtual File System, the Shader System,
the Scenegraph, entity manipulation, layers, sound, dialogs and more.

== Entering Script Code

Script code can be executed by:

. Entering it ad-hoc in the scripting window (Script tab in the properties panel).
. Saving it as a `.lua` file in `install/scripts/` and running it with
  `RunScript yourfile.lua` in the command console.

Scripts are reloaded and re-executed every time `RunScript` is invoked, so
editing and testing is fast.

A comprehensive API demonstration is in `scripts/test.lua`.

== Classes and Methods

=== GlobalSceneGraph

When a map is loaded, the *SceneGraph* is populated with *SceneNodes*
representing brushes, patches, entities and models. Each scenegraph has a
single root.

`SceneNode root()`::
  Returns the root node of the global scenegraph. Can return a NULL node if
  no map is loaded.

=== SceneNode

A SceneNode is the general node object. Actual types are BrushNode, PatchNode,
EntityNode or ModelNode.

`bool isNull()`::
  Returns true if this is an empty NULL node.
`addToContainer(SceneNode newContainer)`::
  Adds this node as a child of the given container.
`removeFromParent()`::
  Removes this node from its parent.
`AABB getWorldAABB()`::
  Returns the absolute bounding box of this node (including children).
`SceneNode getParent()`::
  Returns the parent node (NULL if no parent).
`string getNodeType()`::
  Returns the type string: `"map"`, `"entity"`, `"patch"`, `"brush"`,
  `"model"`, `"particle"`, `"entityconnection"`, `"mergeaction"`, `"unknown"`.
`traverse(function visitor)`::
  Traverses this node and all children. The function receives each node;
  return `true` to continue, `false` to skip children.
`traverseChildren(function visitor)`::
  Traverses only the children of this node.
`setSelected(bool isSelected)`::
  Sets the selection status.
`invertSelected()`::
  Inverts the selection status.
`bool isSelected()`::
  Returns the current selection status.

=== SceneGraph Traversal

Pass a plain Lua function to `traverse()`. Return `true` to continue into
children, `false` to skip them.

[source,lua]
----
-- Skip entity children
GlobalSceneGraph:root():traverse(function(node)
    if node:isEntity() then
        return false  -- don't traverse entity children
    end
    return true
end)
----

The optional *post* visit is not supported in the function-based API. If you
need post-visit behaviour, track state in a local closure.

=== Special SceneNode Types

Every SceneNode exposes the following type-check and cast methods:

* `bool isBrush()` / `BrushNode getBrush()`
* `bool isEntity()` / `EntityNode getEntity()`
* `bool isPatch()` / `PatchNode getPatch()`
* `bool isModel()` / `ModelNode getModel()`

[source,lua]
----
local brush = node:getBrush()
if not brush:isNull() then
    print("Brush with " .. tostring(brush:getNumFaces()) .. " faces")
end
----

==== BrushNode

`int getNumFaces()`::
  Number of faces.
`Face getFace(int faceIndex)`::
  Returns the face (1-based index in Lua).
`bool empty()`::
  True if the brush has no faces.
`bool hasContributingFaces()`::
  True if there are non-degenerate faces.
`removeEmptyFaces()`::
  Removes degenerate faces.
`setShader(string shader)`::
  Sets the shader on all faces.
`bool hasShader(string shaderName)`::
  True if any face uses this shader.
`bool hasVisibleMaterial()`::
  True if any face has a visible shader.
`undoSave()`::
  Saves state to the undo stack before manipulation.
`getDetailFlag()`::
  Returns the detail/structural flag (`BrushDetailFlag.Detail` or
  `BrushDetailFlag.Structural`).
`setDetailFlag(BrushDetailFlag flag)`::
  Sets the detail/structural flag.

==== Face

`undoSave()`::
  Saves current state to the undo stack.
`setShader(string shader)` / `string getShader()`::
  Sets or gets the face shader.
`shiftTexdef(float s, float t)`::
  Shifts the texture.
`scaleTexdef(float s_scale, float t_scale)`::
  Scales the texture.
`rotateTexdef(float angle)`::
  Rotates the texture.
`fitTexture(float s_repeat, float t_repeat)`::
  Fits the texture.
`flipTexture(int axis)`::
  Flips the texture (0 = x-axis, 1 = y-axis).
`normaliseTexture()`::
  Translates the texture towards the origin.
`Winding getWinding()`::
  Returns the winding (Lua array of vertex tables).

==== Winding

The winding is returned as a Lua table (array) of vertex structures.
Each entry has:

`vertex`:: `Vector3` 3D coordinates
`texcoord`:: `Vector2` UV coordinates
`tangent`:: `Vector3`
`bitangent`:: `Vector3`
`normal`:: `Vector3`
`adjacent`:: unsigned index of the adjacent vertex

[source,lua]
----
local winding = face:getWinding()
for _, v in ipairs(winding) do
    print(v.vertex:x() .. "," .. v.vertex:y() .. "," .. v.vertex:z())
end
----

  Insert or remove columns/rows.

  Appends two rows or columns.

  True if the patch has no invalid control points.
`bool isDegenerate()`::
  True if all control vertices are at the same spot.

  True if this is a patchDef3 (fixed tessellation). Note: the first spelling

  Returns x/y subdivision values.

  Sets fixed subdivision.

  Rebuilds the tessellation after control point changes.
`PatchMesh getTesselatedPatchMesh()`::
  Returns a read-only tessellated mesh.
`translateTexture(float s, float t)`::
  Shifts the texture by texel amounts.

==== PatchMesh

`width` / `height`::
  Dimensions of the tessellated matrix.
`vertices`::
  Lua array of `PatchMeshVertex` objects. Access the vertex at column _w_,
  row _h_ as `vertices[h * width + w + 1]` (1-based).

===== PatchMeshVertex

`Vector3 vertex`::
  3D point.
`Vector2 texcoord`::
  UV coordinates.
`Vector3 normal`::
  Normal vector.

===== PatchControl

`vertex`:: `Vector3` control point position.
`texcoord`:: `Vector2` texture coordinates.

===== Subdivisions

`x()`:: Horizontal subdivision count.
`y()`:: Vertical subdivision count.

=== GlobalDeclarationManager

Manages declarations (Materials, Skins, ModelDefs, etc.) from `.mtr`, `.skin`
and `.def` files.

`Declaration findDeclaration(Declaration.Type type, string name)`::
  Finds a named declaration (NULL if not found).
`Declaration findOrCreateDeclaration(Declaration.Type type, string name)`::
  Finds or creates a declaration.
`foreachDeclaration(Declaration.Type type, function visitor)`::
  Visits each declaration: `function(decl)`.
`bool renameDeclaration(Declaration.Type type, string oldName, string newName)`::
  Renames a declaration.
`removeDeclaration(Declaration.Type type, string name)`::
  Removes a declaration from memory (does not touch the source file).
`reloadDeclarations()`::
  Reloads all declarations from disk.
`saveDeclaration(Declaration decl)`::
  Saves the declaration to its file.

==== Declaration.Type

`Declaration.Type.Material`, `Declaration.Type.Table`,
`Declaration.Type.EntityDef`, `Declaration.Type.SoundShader`,
`Declaration.Type.ModelDef`, `Declaration.Type.Particle`,
`Declaration.Type.Skin`

==== Declaration

`bool isNull()`::
  True if this is a NULL object.
`string getDeclName()`::
  Returns the declaration name.
`Type getDeclType()`::
  Returns the declaration type.
`DeclarationBlockSource getDeclSource()`::
  Returns the source block (replaces the old `getBlockSyntax`).
`setDeclSource(DeclarationBlockSource src)`::
  Assigns a new source block.
`string getDeclFilePath()`::
  Returns the mod-relative file path.
`setDeclFilePath(string folder, string filename)`::
  Sets the file location for saving.

==== DeclarationBlockSource

The source block holds raw text content (everything between the outermost
curly braces).

`typeName`:: Type name as in the source file.
`name`:: Declaration name.
`contents`:: Raw inner content (read/write).
`modName`:: Mod/FM name this block was declared in.

[source,lua]
----
local caulk = GlobalDeclarationManager:findDeclaration(
    Declaration.Type.Material, "textures/common/caulk")
local src = caulk:getDeclSource()
print(src.contents)
----

=== GlobalFileSystem

Accesses the Virtual File System (mod folders and PK4 contents).

`forEachFile(string baseDir, string extension, function visitor, int depth)`::
  Traverses VFS files, calling `function(filename)` for each match.
`string findFile(string relativeFilename)`::
  Returns the absolute path.
`string findRoot(string absFilename)`::
  Returns the VFS root for an absolute path.
`string readTextFile(string relativeFilename)`::
  Returns the full text content of a file.
`int getFileCount(string filename)`::
  Returns matching file count in the VFS.

[source,lua]
----
GlobalFileSystem:forEachFile("skins/", "skin", function(filename)
    print("Found: " .. filename)
end, 99)
----

=== GlobalRadiant

Provides convenience methods for finding entities in the active map.

IMPORTANT: These methods return `nil` (not a NULL node) if no map is loaded.
Always check the result before calling methods on it.

`EntityNode findEntityByClassname(string name)`::
  Returns the first entity with the given classname. Returns `nil` or a NULL
  node if not found.
`EntityNode findEntityByName(string name)`::
  Returns the entity with the given `name` spawnarg. Returns `nil` or a NULL
  node if not found.

[source,lua]
----
local ws = Radiant:findEntityByClassname("worldspawn")
if ws and not ws:isNull() then
    ws:setKeyValue("test", "success")
end
----

==== SelectionGroup

`int getId()`:: Returns the group ID.

        print("Member: " .. node:getNodeType())
    end)

----

==== MessageType constants

`Dialog.CONFIRM`:: OK button, notification icon.
`Dialog.ASK`:: YES/NO buttons, question icon.

`Dialog.ERROR`:: OK button, error icon.
`Dialog.YESNOCANCEL`:: YES / NO / CANCEL buttons.

==== Result constants

`Dialog.CANCELLED`, `Dialog.OK`, `Dialog.NO`, `Dialog.YES`

[source,lua]
----
-- Simple message box
GlobalDialogManager:createMessageBox(
    "Info", "Operation complete.", Dialog.CONFIRM):run()

-- Dialog with inputs
local dialog = GlobalDialogManager:createDialog("Export Settings")
local fileHandle = dialog:addEntryBox("Filename:")
local options    = { "OBJ", "ASE" }
local fmtHandle  = dialog:addComboBox("Format:", options)

if dialog:run() == Dialog.OK then
    local filename = dialog:getElementValue(fileHandle)
    local format   = dialog:getElementValue(fmtHandle)
    print(filename .. " (" .. format .. ")")
end
----
